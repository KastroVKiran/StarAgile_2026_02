=> File Permissions in Linux
File 			----> rw-r--r--
Directory	----> rwxr-xr-x

any file permissions in Linux is categorized into 3 groups;
	user (u)
	group (g)
	others (o)

For each category, we will have 3 permissions;
	read (r)
	write (w)
	execute (x)

root.txt
rw-r--r--

u ---> read, write
g ---> read
o ---> read

to modify the file permissions, we have "chmod"
chmod u+x kastro.txt
chmod g+wx kastro.txt
chmod o+wx kastro.txt

chmod u-x kastro.txt
chmod o-rwx kastro.txt

File permissions using numbers;
0 - no permissions
1 - execute
2 - write
3 - 2+1 - write + execute
4 - read
5 - 4+1 - read +execute
6 - 4+2 - read + write
7 - 6+1 - read + write + execute
