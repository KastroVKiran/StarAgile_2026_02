What is an OS?
OS is a software which is used to communicate with the machines (Laptop/Desktop/Mobile/Physical Server/Virtual Machine)
OS is going to acts an intermediate layer between user and the machine
OS is mandatory to work with any machines
It is going to provide an environment to work with the machines

Types of OS;
Windows
MacOS
Linux OS
....

As a software engineer, it is important to know about Linux OS

Windows OS Vs Linux OS

Ubuntu				- ubuntu
